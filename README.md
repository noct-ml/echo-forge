### **Clean Markdown**


### **Speaker Separated, Obsidian Ready**

```bash

  --by-speaker --user-label "James" --pretty-md \
  --max-width 90 --theme obsidian --obsidian-links
```

```bash

  --by-speaker --pretty-md --toc-depth 3
```

```bash

```
